# social-media

Social Media Web App • Live Link: https://social-media-web-react.vercel.app/

• Tech Stack: Java, Spring Boot, JavaScript, React, Redux, Bootstrap, MySQL, Hibernate, JUnit, React Testing Library, Jest, Vercel, Railway & PlanetScale.

•	Implemented social media functionalities such as real-time text & image posting, profile picture, public user profiles, profile updates & user lists.

•	Implemented secure login & signup using Spring Security & storage for image on the backend server. Applied Test-Driven Development with the unit test code coverage of 77% in the backend & 93% in the frontend which can improve code quality & bug reduction up to 30%. Deployed the Spring Boot app on Railway & the React app on Vercel. Used PlanetScale for the MySQL Database.
